# SQL Intern Training – Task 1

## Project Description
This project demonstrates basic SQL operations including:
- Database creation
- Table creation
- Data insertion
- Data retrieval using SELECT queries

## Tools Used
- MySQL Community Server
- MySQL Workbench

## Files Included
- intern_training_db.sql – Full database export
- task1_sql_basics.sql – SQL practice queries

## How to Run
1. Open MySQL Workbench
2. Create a new connection
3. Run the SQL file using File → Open SQL Script
